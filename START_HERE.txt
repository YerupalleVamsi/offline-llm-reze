❓ QUICK TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

"Backend not responding"
  → Check: http://localhost:8000/health
  → If it doesn't load, backend not running
  → Restart RUN_UNIFIED.bat

"Failed to fetch"
  → Check terminal shows "Uvicorn running..."
  → Wait for build to complete (1-2 min first time)
  → Restart browser (Ctrl+Shift+R)

"Port 8000 already in use"
  → Close other apps using port 8000
  → Or edit script to use port 8001

"FFmpeg not found"
  → Audio won't work
  → Download: https://ffmpeg.org/download.html

For more help → Read QUICK_START.txt
