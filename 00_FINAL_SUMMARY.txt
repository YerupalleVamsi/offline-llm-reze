## 🆘 IF ISSUES OCCUR

### "Command not found"
→ Make sure you're in the right folder: `D:/testing shit`

### "Failed to fetch"
→ Check: http://localhost:8000/health
→ Should show: `{"status":"ok","message":"..."}`

### "Port already in use"
→ Close other apps using port 8000
→ Or edit script to use different port (8001)

### "Frontend won't load"
→ Check terminal for build errors
→ Verify `frontend/dist/` folder exists

### "FFmpeg not found"
→ Audio won't work
→ Download: https://ffmpeg.org/download.html

**For more details:** See `QUICK_START.txt`
