## 🚀 What Happens When You Run

### Step 1: Frontend Build
```
npm install --legacy-peer-deps
npm run build
```
Creates `frontend/dist/` with optimized production files.

### Step 2: Backend Start
```
python -m pip install -r requirements.txt  (first time)
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```
Backend starts and mounts the frontend files.

### Step 3: Access
```
Browser → http://localhost:8000
         ├─ Frontend UI loads
         ├─ API calls work (no proxy needed)
         └─ Everything on same port ✅
```

## 💡 Pro Tips

### Different Port
Edit the batch file to use port 8001 instead of 8000 if you want.

### Development with Hot Reload
If you want React hot reload during development:
```bash
# Terminal 1: Backend
cd backend && python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

# Terminal 2: Frontend dev server
cd frontend && VITE_API_URL=http://127.0.0.1:8000 npm run dev
```
Then access: http://localhost:5173

### Production Deployment
The `RUN_UNIFIED.bat` setup is exactly what you'd deploy to a server.
